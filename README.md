## 📂 Projects

### 🧠 Customer Churn Prediction — Neural Network
A neural network that predicts whether a bank customer will leave (churn), trained on the Churn Modelling dataset.
- **Workflow:** feature scaling, categorical encoding, network design, training, and performance analysis
- **Tech:** Python, TensorFlow/Keras, Pandas, NumPy
- 📄 `Neural Network on Churn_Modelling.ipynb`

### 📊 Startup Profit Prediction — Regression
A regression model that predicts startup profit based on R&D, administration, and marketing spend.
- **Workflow:** EDA, handling categorical features, feature selection, model training and evaluation
- **Tech:** Python, scikit-learn, Pandas
- 📄 `Machine learning_Startup.ipynb`

### 🚢 Titanic Survival Prediction — Classification
A classification model predicting passenger survival — a complete end-to-end ML exercise.
- **Workflow:** data cleaning, handling missing values, feature engineering, model evaluation
- **Tech:** Python, scikit-learn, Pandas
- 📄 `ML on Titanic Dataset.ipynb`

### 🎮 Gesture-Based Game Control — Computer Vision
A real-time hand-gesture recognition system that controls game actions through a webcam — no physical controller needed.
- **Workflow:** live video capture, hand tracking, gesture classification, low-latency input mapping
- **Tech:** Python, OpenCV
- 📄 `GAME control using GESTURE.ipynb`
- 🔗 [LinkedIn demo post](https://www.linkedin.com/feed/update/urn:li:activity:7188188300409483264)

### 🚀 DOSS Capstone — Multi-Feature AI System
The final project from my DOSS training program, integrating several real-world features into a single Python system:
- Snapchat-style camera filters
- Nearby location search using geographical coordinates
- Automated email sending
- Listing running AWS EC2 instances via IAM credentials
- AI chatbot through API integration
- Gesture recognition with OpenCV
- Website creation and deployment on AWS EC2, served over a public IP
- **Tech:** Python, OpenCV, AWS (EC2, IAM), REST APIs

## 🚀 How to Run

```bash
# Clone the repository
git clone https://github.com/Shivansh-sa/Projects.git
cd Projects

# (Recommended) create a virtual environment
python -m venv venv
source venv/bin/activate        # On Windows: venv\Scripts\activate

# Install dependencies
pip install numpy pandas scikit-learn tensorflow opencv-python matplotlib jupyter

# Launch Jupyter
jupyter notebook
```

Open any `.ipynb` file and run the cells top to bottom.
